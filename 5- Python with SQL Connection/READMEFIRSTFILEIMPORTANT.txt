1- run take_user_input.py
2- edit the directory in the code to save the files where you want
3- run database.sql to create the database
4- change the direction in the user_data_script.py
5- change the server name to the server of yours locally
6- run the user_data_script.py
7- check the user_data.txt to see the data you enter in the take_user_input.py
8- check the table you create in the database to see the data stored

NOTE: everytime you enter data in the take_user_input.py file, you need to run the user_data_script.py
It's not automated
